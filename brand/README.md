# ATOCODE Brand Assets

Use these SVG files as the source logo assets for web and social platforms.

- `atocode-mark.svg` - primary icon mark for the website, favicon, small placements, and app-style usage.
- `atocode-mark.png` - PNG export of the primary icon mark.
- `atocode-lockup.svg` - horizontal logo lockup for banners, proposals, email signatures, and larger brand placements.
- `atocode-lockup.png` - PNG export of the horizontal lockup on a dark background.
- `atocode-social-avatar.svg` - square profile/avatar artwork for Instagram, Facebook, LinkedIn, and other platforms.
- `atocode-social-avatar.png` - ready-to-upload square PNG avatar for social profiles.

The mark combines a geometric `A` monogram with code brackets to keep the brand tied to development, UI, hosting, and maintenance.
